How to Install

1

Download

Download the latest .zip from GitHub Releases.

2

Load Unpacked

In Chrome, go to Extensions, enable Developer Mode, then click Load unpacked.

3

Play Music

Open YouTube Music and click the K-ROOM icon in the player bar.
